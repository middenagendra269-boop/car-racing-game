# Car Racing Game 🏎️

Text-based Python game built on Redmi tablet using Pydroid 3.

### How to play
1. Download `car_game.py`
2. Open in Pydroid 3
3. Run ▶️
4. Type `l` or `r` to dodge X obstacles

### Score
Best score: 40  
Goal: Survive 10 rounds = 100 points

### Built with
- Python 3
- `random` and `time` modules
